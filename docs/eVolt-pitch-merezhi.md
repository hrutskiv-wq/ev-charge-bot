# eVolt — підключення для мережі станцій
### Оператор із багатьма станціями / розумними зарядками

**Єдиний канал залучення водіїв.** Ваші станції з'являються в боті-агрегаторі,
де водії шукають зарядку на одній живій мапі — **без встановлення вашої окремої апки**.
Ви лишаєте свій застосунок; ми — шар агрегації попиту зверху.

---

## Головне про гроші
Оплата йде **напряму на ваш еквайринг** (Monobank). eVolt не стає власником ваших
коштів. Комісія платформи — **5%** з проведених через неї транзакцій.

## Що ви отримуєте
- **Додатковий канал залучення** — водії знаходять вас без вашої апки.
- **Видимість на єдиній живій мапі** станцій: що більше водіїв у боті — то цінніша присутність.
- **Готовність до роумінгу через OCPI** — один стандарт замість зоопарку інтеграцій.
- **Масове підключення** станцій через API/OCPI, зведена звітність по мережі.
- Для розумних зарядок — **точний розрахунок за кВт·год** (без переплат і повернень).

## Оплата й повернення
- **Лічильникові / розумні зарядки:** розрахунок за фактом — переплати немає,
  повернення практично не потрібні.
- **Де є передоплата:** часткове повернення через Monobank підтримується
  (невикористане повертається на картку водія).

## Інтеграція
- **OCPI-first:** підключаємо ваші станції за стандартом (локації, живий статус, тарифи).
- Кастомний адаптер — лише як виняток для великого оператора без OCPI.
- Живий статус наповнює єдину мапу — цінність і для водіїв, і для вас.

## Модель співпраці
**5% комісії платформи.** Обсяги, умови інтеграції та пріоритет у видачі — індивідуально,
залежно від розміру мережі й готовності ділитися живими даними станцій.

> Ви не втрачаєте свій бренд і апку — ви додаєте канал попиту й місце на спільній мапі,
> куди водії приходять першими.
